# Education Provider Registry Data

This site publishes generated documentation views over the data-modelling artefacts in this repository.

- [Vocabulary](./vocabulary/)
- [Taxonomy](./taxonomy/)
- [Ontology](./ontology/)

The vocabulary pages are generated from `models/establishment-details-vocabulary-skos.ttl` during the GitHub Pages build.

The taxonomy page is generated from `models/establishment-details-taxonomy-skos.ttl` during the GitHub Pages build.

The ontology page is generated from `models/establishment-details-conceptual-model.ttl` by WIDOCO during the GitHub Pages build.

The Turtle files remain the source of truth.
